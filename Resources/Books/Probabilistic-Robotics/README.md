# Probabilistic Robotics — Thrun, Burgard, Fox

## Why This Book
Probabilistic Robotics is the foundational reference for 
state estimation, localization, mapping, and SLAM in 
autonomous systems. Every major algorithm in modern 
robotics — Kalman Filter, EKF, Particle Filter, AMCL, 
graph SLAM — traces its mathematical foundations to 
this book.

This is not a book to read cover to cover.
It is a reference to consult exactly when the engineering 
requires it.

## What This Book Covers
- Probabilistic foundations of robotics
- Gaussian filters: Kalman Filter, Extended Kalman Filter
- Non-parametric filters: Particle Filter, Histogram Filter
- Robot motion models
- Sensor models (beam model, likelihood field)
- Mobile robot localization (Markov, AMCL)
- Occupancy grid mapping
- Simultaneous Localization and Mapping (SLAM)
- Graph-based SLAM and pose graph optimization
